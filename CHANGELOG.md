# Changelog

All notable changes to this project will be documented in this file.

## Unreleased

- Changed `codex-harness init` to install repository skills to `.agents/skills` by default.
- Changed `codex-harness init` to require explicit `--all` or `--skill`.
- Added `--skill-layout legacy|both` for migration from the older root `skills/` layout.
- Changed user-level sync defaults to `~/.agents/skills`.
- Added `codex-harness marketplace` for generating a local Codex plugin marketplace catalog.
- Added CLI reference, compatibility documentation, and concrete example scenarios.
- Expanded CLI tests for official skill layout, legacy compatibility, and marketplace generation.
- Added package metadata links for documentation and changelog.

## 0.1.0 - 2026-06-26

- Added cross-platform `codex-harness` CLI.
- Added Python package metadata.
- Added public-facing README quickstart and demo path.
- Added plugin packaging, architecture, demo, and roadmap docs.
- Added validated examples for Python APIs, React apps, and OSS maintainer workflows.
- Added GitHub Actions CI, issue templates, PR template, contributing guide, and security policy.
