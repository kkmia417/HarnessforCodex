# Triage Rubric

Use these classifications:

- `bug`: reproducible product defect
- `feature`: new capability or behavior change
- `support`: usage question or environment-specific help
- `docs`: missing, stale, or confusing documentation
- `security`: vulnerability or secret exposure
- `release-blocker`: defect that should stop or delay a release

Every triage response should include the likely label, missing information, and
the next maintainer action.
