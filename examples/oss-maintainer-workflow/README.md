# OSS Maintainer Workflow Example

This example shows a small harness for issue triage and release preparation in
an open-source project.

Try it:

```bash
codex-harness validate examples/oss-maintainer-workflow
```

Prompt:

```text
Use $maintainer-triage to classify this issue and propose the next maintainer action.
```
